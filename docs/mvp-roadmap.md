# MVP Roadmap

## Completed in this scaffold

- FastAPI backend with health, analyze, and feedback endpoints
- Bangla normalization, lexicon spell checking, conservative rules, and suggestion merging
- Tiptap web editor with issue marks and feedback actions
- Chrome extension skeleton with backend integration and overlay indicators
- SQLite feedback logging
- sample fixtures, tests, corpus scripts, and ML scaffolding

## Next milestones

1. Expand the Bangla lexicon and add personal dictionary persistence.
2. Improve plain-text to rich-text span mapping for multiline documents and more complex editors.
3. Build an annotated dataset pipeline from real feedback and curated Bangla corpora.
4. Train the first custom tokenizer and baseline detector model from scratch.
5. Add precision dashboards for offline fixture evaluation and regression tracking.
